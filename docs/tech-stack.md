# 技術スタック

> ※ 技術スタックは現在選定中。確定次第このドキュメントを更新する。

## 構成概要

フロントエンド / バックエンドAPI / データベース の3層構成。

```
[ ブラウザ ]
    ↓ HTTP
[ バックエンドAPI ]
    ↓ ORM / SQL
[ データベース ]
```

## 選定基準

技術を選定する際は以下の観点を考慮する。

| 観点 | 内容 |
|---|---|
| 学習目的への適合 | スクール課題の学習ゴール（フルスタック開発経験）に沿っているか |
| セットアップの容易さ | ローカル環境で素早く構築できるか |
| ドキュメントの充実 | 公式ドキュメントや参考資料が豊富か |
